# Flota 360

es una plataforma web diseñada para centralizar y optimizar la gestión de flotas vehiculares dentro de cualquier empresa.
Permite administrar vehículos, almacenar documentos, recibir alertas automáticas de vencimiento y mantener la información organizada, segura y disponible para equipos administrativos y operativos.

El sistema está construido para ser escalable, multiempresa y accesible desde cualquier dispositivo, con el objetivo de transformarse en un producto vendible y adaptable a múltiples compañías.
